# 原始样本池

原始样本池保存后台上传或收集到的图片样本。样本可以来自管理员上传、用户仿图沉淀或链接收集。

主流程里，管理员上传时只判断图片是否值得学习，不需要手动选择平台、品类、图片类型或风格名。系统会把这些图保存为候选占位信息：

- `platform=free`
- `category=general`
- `imageType=style_candidate`
- `styleName=待 Codex 分析`

样本字段包括线上样本 ID、图片地址、来源类型、来源备注、当前占位标签、当前状态和基础分析信息。占位标签只用于导出和追踪，不代表最终风格判断。

样本状态：

- `pending_analysis`：等待分析或等待管理员处理。
- `pending_review`：已进入审核待确认。
- `approved`：管理员认为值得学习，可进入候选批次。
- `rejected`：不进入后续风格学习。

通过样本不会自动生成首页经典风格。样本必须先进入候选批次，再经过离线 Codex Skill 自动聚类、导入、审核、发布。
