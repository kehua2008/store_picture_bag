# 原始风格候选池

原始风格候选池用于收集管理员认为值得学习的优质图片。候选阶段不要求管理员选择最终品类、图片类型、风格名或商品简介。

管理员上传一批优质图片后，系统会自动创建候选批次。批次内图片通常带有统一占位标签 `free/general/style_candidate/待 Codex 分析`。这些标签不参与最终分类，只用于说明图片尚未经过 Codex 聚类。

候选批次导出后，Codex Skill 负责把图片自动归到经典风格组下，例如多巴胺、老钱质感、韩系松弛、城市通勤、山系机能，或根据图片发现更多稳定风格。每个风格组背后保留若干样本图片，并由这些图片总结通用提示词。

候选批次字段：

- `id`：候选批次 ID。
- `name`：管理员命名的批次名。
- `sampleIds`：本批次包含的线上样本 ID。
- `sourceNote`：批次备注。
- `status`：批次状态。
- `createdAt` / `updatedAt`：时间戳。
- `exportedAt` / `analyzedAt` / `importedAt`：流程节点时间。

候选批次状态：

- `collecting`：继续收集图片。
- `exported`：已导出本地分析包。
- `analyzed`：本地已完成分析，等待导入。
- `imported`：分析结果已导入为正式风格板草稿。
- `archived`：批次归档。
