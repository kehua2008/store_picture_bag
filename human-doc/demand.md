唯品会商家上架产品图片完整规则（2026最新，VOP开放平台官方标准）
 
唯品会商品图片分为主图（轮播图）、SKU色图、白底图、详情页图、场景图、视频封面图6大类，以下为尺寸、格式、大小、内容规范、禁止项、各类目特殊要求全明细，严格按此上传可避免驳回、上架失败、流量受限。
 
一、通用基础硬性规则（所有图片必须遵守）
 
1. 格式：仅支持 JPG/JPEG，禁止PNG、GIF、TIFF；色彩模式必须RGB，禁止CMYK（会变色）
2. 分辨率：72dpi，图片清晰无噪点、无拉伸变形、无模糊、无压缩失真
3. 文件大小：单张≤2MB（主图/色图≤1MB；详情图单张≤1.5MB）
4. 水印/文字：全程禁止水印、促销文字、价格、二维码、网址、品牌LOGO、活动标语、拼接图、边框
5. 光影：禁止大面积黑阴影、强光反光、环境杂物反射，色彩真实还原实物
6. 主体：商品居中、完整展示，长边距画布边框≤10px；展示商品与销售规格、颜色、数量完全一致
 
 
 
二、主图（轮播图，商品首屏展示，核心流量入口）
 
1. 尺寸&比例（2024.9.1起强制执行，不合规直接上传失败）
 
- 服装/鞋包类目：竖版长方形，宽1340px × 高1785px，比例3:4，宽≥1340、高≥1785，不可正方形
- 非服装类目（美妆、家居、3C、食品、家纺、百货等）：正方形，1200×1200px（最小800×800px），1:1比例
 
2. 数量要求
 
- 最少5张，最多8张；顺序固定：
1图：正面全貌图/场景主图
2图：侧面/背面全貌
3-4图：细节特写（面料、五金、logo、做工）
5图：纯白底图（必传，用于搜索流量）
 
3. 内容规范
 
1. 第1张主图：服装优先模特实拍场景图；标品/家居可实景图，禁止纯白底
2. 禁止：多件商品同框、无关道具、背景杂乱、文字营销信息
3. 商品必须完整展示，不可裁切主体
 
 
 
三、SKU色图（规格图，对应每个颜色/尺码）
 
1. 尺寸：正方形 800×800px及以上，建议1000×1000px，1:1比例
2. 格式/大小：JPG，单张≤500KB
3. 内容：纯实物白底图，对应SKU实际颜色，无色差；禁止场景图、模特图、多色拼接
4. 要求：1个SKU对应1张色图，不可混用
 
 
 
四、白底图（搜索流量必备，必传）
 
1. 硬性标准
 
- 尺寸：1200×1200px（最小800×800px），正方形
- 背景：RGB 255,255,255绝对纯白，无阴影、无渐变、无抠图痕迹、无背景杂物
- 主体：完整居中，占画布70%-85%，不可裁切
- 禁止：LOGO、文字、水印、模特、场景、阴影、倒影
 
2. 用途：唯品会搜索页、推荐页、活动会场优先抓取，无白底图流量大幅降低
 
 
 
五、详情页图片（宝贝描述图，核心转化）
 
1. 尺寸：固定宽度750px，高度不限（单张高度建议≤3000px，避免加载卡顿），竖版排版
2. 格式/大小：JPG，单张≤1.5MB，整组详情图≤10MB
3. 内容结构（建议）：- 首屏：品牌海报、产品整体卖点
- 第二部分：细节特写、材质、尺码表、参数
- 第三部分：场景使用图、售后保障、注意事项
4. 禁止：横向长图、模糊图、重复图、营销大字、违规宣传词
 
 
 
六、场景图/模特图（主图可用）
 
1. 尺寸：服装3:4（1340×1785），标品1:1（1200×1200）
2. 背景：干净简约，突出商品，背景元素≤4种，不可抢主体风头
3. 模特：服装类目优先真人模特，姿势自然，不遮挡商品；禁止低俗、过度美颜、过度修图
 
 
 
七、各类目特殊强制规则（重点避坑）
 
1. 服饰鞋包
 
- 主图必须3:4竖版，禁止正方形；首图优先模特上身图
- 细节图必须展示：领口、袖口、走线、面料、吊牌
 
2. 美妆护肤
 
- 主图正方形1200×1200；禁止过度美颜、虚假效果图
- 必须上传成分、质地、上脸/上手实拍图
 
3. 家居家纺/家具
 
- 主图800×800起；实景图可接受，禁止纯白底首图
- 展示整体+细节（面料、尺寸、做工），必须与实物一致
 
4. 3C数码/家电
 
- 主图正方形；必须展示实物全貌、接口、配件，禁止网图、盗图
- 白底图必须清晰展示机身完整形态
 
 
 
八、绝对禁止项（违规直接驳回/下架/扣分）
 
1. 盗图、网图、非实拍图、侵权图片
2. 水印、文字、二维码、营销标语、价格信息、第三方LOGO
3. 拼接图、多商品同框、无关杂物、低俗背景
4. 色差严重、修图过度、虚假宣传（夸大功效）
5. CMYK格式、PNG/GIF格式、尺寸不达标、文件过大
 
 
 
九、快速自查清单（上传前核对）
 
1. 主图尺寸：服装3:4，其他1:1，像素达标
2. 必传5张主图+1张白底图，SKU对应色图
3. 无水印、无文字、无拼接、无违规内容
4. 格式JPG，RGB模式，72dpi，大小合规
5. 详情图宽750px，排版清晰
