# 待启动计划归档

这个目录用于保存“后期可能启动实施”的产品、商业和技术计划。以后需要启动某个方向时，先从这里检索对应计划，再转成 OpenSpec change 或具体开发任务。

## 使用规则

- 这里只保存待启动或暂缓的计划，不代表当前已经进入实施。
- 每个计划文件都需要包含统一字段，方便后续快速接手。
- 计划启动后，把状态从“待启动”改为“已启动”，并补充实施入口或对应变更文档。
- 计划完成后，把状态改为“已完成”，并补充最终落地位置。
